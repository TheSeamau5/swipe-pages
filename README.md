## Installation
With Bower:
```cli
bower install swipe-pages
```

## Basic Example
```html
<swipe-pages>
	<swipe-page>I am page 0<swipe-page>
	<swipe-page>I am page 1<swipe-page>
	<swipe-page>I am page 2<swipe-page>
</swipe-pages>
```

## Events

* swipeStart
* swipeEnd - event.detail.validTarget {Boolean}
